# Exercise Index

## Downloadable/Customizable Indexes
App has downloadable indexes for the different exercise types. This should be chosen by the user to prevent them from having unnecessary bloat. 

## Tags
It might be pertinent to utilize tags for things as opposed to making them separate exercises. 

Such as:
- Incline (Incline, Flat, Decline, or in degrees), 
- Grip-Width (Wide/Narrow), 
- ROM (Full,  w/Deficit, Lengthened/the-opposite-of-that Partials)
- Weight Type

	- Barbell
	- Smith Machine
	- Pin-Type
		- Stack-loaded
		- Plate-loaded
	- Bodyweight
		- Weighted
