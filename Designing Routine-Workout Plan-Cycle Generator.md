# Time Crunch Tips (<1hour)
-   Use mostly compounds
    -   Mostly general (Normal instead of Wide BP)
-   Use exercises that use less warm up times, lunge vs hip thrust.
-   Lighter / higher rep ranges, perhaps counter intuitively so that less warm up sets. Heavy vs light.
-   Antagonist supersets, drops, myos, myo-match sets.
    -   Myo-match sets for: biceps, calves, side/rear delts, hams/quads in isolation (curl/ext.), lat prayers, pushdowns, etc.
-   Lower RIR
-   Don’t progress in volume or only add some sets. 8-12 instead of 16.
-   Specialization phases.
